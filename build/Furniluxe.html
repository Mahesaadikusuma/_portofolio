Furniture Laravel

Pengembangan Website Furniture jual beli barang-barang Furniture yang dibuat dengan menggunakan html, css, javascript, laravel, serta penerapan Sass sebagai praprosesor CSS. Dalam proyek ini, Anda juga memanfaatkan library animasi, khususnya Swiper JS, untuk menambahkan elemen dinamis ke antarmuka

Laravel sebagai kerangka kerja backend menegaskan pemahaman Anda tentang pengembangan web berbasis server-side. Ini mencerminkan kecakapan Anda dalam mengelola logika bisnis dan keamanan aplikasi. Menggunakan fitur Login laravel nyaitu Jetstream untuk mempermudah melakukan authentication dan authorization untuk keamanan dalam Website. Juga melakukan middlware untuk mengetahui Admin dan user

Hal-Hal menggunakan Laravel

  • Jetstream
  • Laravel Blade
  • Database Migration
  • Model
  • View
  • Controller
  • Middlware
  • Routing
